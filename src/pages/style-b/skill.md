# Intent
에디토리얼/포스터 스타일의 대담하고 시각적으로 인상적인 디자인. 강한 대비와 과감한 타이포그래피로 시각적 임팩트를 주며, 넉넉한 여백과 레이어드 카드 스타일로 현대적이고 세련된 느낌을 연출한다. 크리에이티브 분야나 마케팅 대시보드에 적합한 스타일.

# Style Rules
- 라운드 코너를 크게 사용한다 (16-24px)
- 그림자를 적극적으로 활용하여 깊이감을 준다
- 강한 색상 대비를 사용한다 (흑백 + 강조색)
- 타이포그래피를 과감하게 사용한다 (큰 제목, 굵은 웨이트)
- 여백을 넉넉하게 활용한다
- 버튼과 배지에 bold한 스타일을 적용한다
- 카드는 떠있는 느낌의 그림자를 사용한다
- 호버 시 스케일 변화나 그림자 증가 효과를 준다
- 강조색(accent)을 전략적으로 사용한다
- 상태 배지는 채워진 스타일로 눈에 띄게 표현한다

# Tokens
- color.primary: black
- color.accent: violet-600
- color.success: emerald-500
- color.warning: orange-500
- color.error: rose-500
- radius.md: 16px
- radius.lg: 24px
- space.section: 32px
- space.element: 16px
- shadow.card: 0 10px 40px rgba(0,0,0,0.1)

# Do
- 과감한 시각적 표현을 사용한다
- 여백을 넉넉히 활용한다
- 그림자와 레이어로 깊이감을 준다
- 강조 요소는 눈에 띄게 표현한다
- 호버 인터랙션에 변화를 준다

# Don't
- 너무 많은 요소를 빽빽하게 배치하지 않는다
- 약한 대비나 흐릿한 스타일 사용 금지
- 작고 눈에 띄지 않는 버튼 금지
- 일관성 없는 radius 혼용 금지
- 그림자 없이 플랫하게만 표현하지 않는다

# Dark Mode
- 다크 모드에서 그림자를 더 강화하여 깊이감을 극대화한다
- 카드의 떠있는 느낌을 강조하기 위해 어두운 배경에 더 강한 그림자를 사용한다
- 강조색(accent)의 밝기를 다크 모드에 맞게 조정하여 대비를 유지한다
- 배지는 밝은 색상으로 눈에 띄게 하되 텍스트 대비를 확보한다
- 글래스모피즘(backdrop-blur) 효과를 활용하여 레이어드 느낌을 강화한다
- 호버 시 그림자 변화를 더 극적으로 표현한다

# Notes
시각적 임팩트가 중요한 크리에이티브 도구나 마케팅 대시보드에 적합. 넉넉한 여백과 대담한 타이포그래피로 현대적이고 트렌디한 느낌을 준다. 정보 밀도보다는 시각적 경험을 우선시한다.
