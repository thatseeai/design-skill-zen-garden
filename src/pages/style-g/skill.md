# Intent
전통적인 CSS Zen Garden에서 영감을 받은 선(禪) 스타일의 평화롭고 고요한 디자인. 자연과 조화를 이루는 동양적 미학을 현대 웹 UI에 적용하여, 차분하고 명상적인 분위기를 연출한다. 대나무, 돌, 물결 등 자연 요소를 시각적 모티프로 활용한다.

# Style Rules
- border-radius는 0으로 유지하여 날카롭고 깔끔한 선을 강조한다
- 세리프 폰트(Crimson Text, Noto Serif JP)를 사용하여 전통적인 느낌을 준다
- 색상 팔레트는 자연에서 영감을 받는다: 대나무 녹색(#4a6c45), 주홍(#9e2a2b), 돌색(stone)
- 카드와 컨테이너는 반투명 배경(white/60~90)으로 배경과 어우러지게 한다
- 그림자는 최소화하고, 보더로 요소를 구분한다
- 테이블 헤더에 연한 녹색 배경을 사용한다
- 탭과 네비게이션은 하단 보더 강조 스타일을 사용한다
- 장식적 배경 이미지(리본, 대나무)를 레이어로 배치한다
- 여백은 넉넉하게 사용하여 여유로운 느낌을 준다
- 호버 상태는 자연스러운 색상 전환으로 표현한다

# Tokens
- color.primary: #9e2a2b (주홍)
- color.secondary: #4a6c45 (대나무 녹색)
- color.accent-dark: #2c4a29 (짙은 녹색)
- color.sand: #d4a373 (모래/흙 톤)
- color.background: #dceaf3 (하늘색)
- color.paper: rgba(255, 255, 255, 0.7)
- font.serif: 'Crimson Text', 'Noto Serif JP', serif
- radius: 0px
- space.section: 48px
- space.element: 24px

# Do
- 자연에서 영감을 받은 색상 조합을 사용한다
- 세리프 폰트로 전통적인 분위기를 연출한다
- 반투명 배경으로 레이어 효과를 활용한다
- 섹션 제목에 장식적인 하단 보더를 추가한다
- 여백을 충분히 사용하여 숨 쉴 공간을 만든다

# Don't
- 둥근 코너 사용 금지 (radius는 항상 0)
- 과도한 그림자나 입체 효과 사용 금지
- 원색이나 네온 컬러 사용 금지
- 바쁘거나 혼잡한 레이아웃 지양
- 장식 요소가 콘텐츠를 방해하지 않도록 한다

# Notes
CSS Zen Garden의 정신을 계승하여 "동일한 HTML, 다른 CSS"의 가능성을 보여주는 디자인. 동양적 선(禪) 철학의 고요함과 균형을 웹 인터페이스로 표현했다. 배경 이미지와 장식 요소는 z-index를 조절하여 콘텐츠 가독성을 해치지 않으면서 분위기를 연출한다.
