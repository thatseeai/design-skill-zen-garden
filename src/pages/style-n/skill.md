# Intent
럭셔리 에디토리얼 무드의 하이엔드 매거진 인터페이스를 구현한다. 강한 헤드라인 계층, 넓은 여백, 절제된 컬러(블랙/아이보리/브론즈)로 브랜드 품격을 전달하면서도 데이터 대시보드의 가독성과 조작성은 유지한다.

# Style Rules
- 타이포 계층을 극대화한다: 섹션 제목은 크게, 본문은 정돈된 리듬으로 배치한다
- 컬러는 최대 3개 축으로 제한한다 (잉크 블랙, 아이보리, 브론즈 포인트)
- 배경은 단순 단색이 아니라 고급 재질감(그라데이션+텍스처)으로 구성한다
- 카드/테이블/폼은 얇고 정교한 보더로 분리하고 과한 라운드는 피한다
- CTA는 핵심 액션만 브론즈 강조, 나머지 버튼은 톤다운한다
- 넓은 여백과 명확한 섹션 분리로 시선 흐름을 만든다
- 장식은 선(line)과 미세한 글로우 중심으로 최소화한다
- 다크/라이트 모드 모두 동일한 에디토리얼 분위기를 유지한다

# Tokens
- color.ink: #111111
- color.ivory: #f6f1e8
- color.bronze: #b6905d
- color.graphite: #2d2a27
- color.mist: #d9d0c3
- radius.base: 6px
- border.fine: 1px solid rgba(182, 144, 93, 0.45)
- shadow.soft: 0 10px 24px rgba(17, 17, 17, 0.16)

# Do
- 헤드라인과 숫자에 충분한 대비를 확보한다
- 본문/보조 텍스트가 배경에 묻히지 않도록 명도 차를 유지한다
- 컴포넌트 전반에 같은 보더/간격 체계를 적용한다
- 모바일에서도 여백 리듬이 유지되도록 패딩을 단계적으로 조절한다
- 표/폼/탭은 장식보다 정보 전달을 우선한다

# Don't
- 고채도 다색 포인트를 동시에 사용하지 않는다
- 두꺼운 보더나 과한 그림자로 럭셔리 톤을 깨지 않는다
- 모든 버튼을 동일 강도로 강조하지 않는다
- 배경 텍스처 대비를 높여 텍스트 가독성을 해치지 않는다
- 섹션 간 간격을 좁혀 에디토리얼 리듬을 망치지 않는다

# Notes
핵심은 "절제된 화려함"이다. 디자인 포인트는 최소화하고, 타이포와 여백의 밀도로 프리미엄 감각을 만든다.
