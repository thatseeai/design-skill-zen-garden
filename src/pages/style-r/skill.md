# Intent
오가닉 웰니스 감성의 차분하고 건강한 인터페이스를 구현한다. 흙, 식물, 크림 톤을 기반으로 유기적 곡선과 부드러운 깊이감을 만들고, 자연 재질(종이/섬유)의 촉감을 담아 편안하고 신뢰감 있는 대시보드 경험을 제공한다.

# Style Rules
- 팔레트는 흙/식물/크림 톤 중심으로 제한한다 (고채도 원색 남용 금지)
- 카드/컨트롤은 유기적 곡선 라운드와 소프트 섀도우를 일관 적용한다
- 배경은 종이/섬유 질감이 느껴지는 그라데이션+텍스처 레이어로 구성한다
- 구조는 단순하게 유지하되 섹션 흐름은 넓은 여백으로 명확히 분리한다
- 버튼 계층은 1차 CTA만 식물 계열로 강조하고 나머지는 톤다운한다
- 테이블/폼은 장식보다 정보 가독성을 우선해 대비를 충분히 확보한다
- 장식 요소는 화면 주변부에 저강도로 분산 배치한다
- 다크/라이트 모드 모두 같은 웰니스 무드를 유지한다

# Do
- 큰 섹션 제목과 충분한 여백으로 시선 흐름을 만든다
- 본문/숫자/라벨의 명도 대비를 확보해 읽기 쉽게 유지한다
- 배경 텍스처는 은은하게 적용해 콘텐츠를 방해하지 않게 한다
- 상태 배지는 색상 의미가 즉시 전달되도록 명확히 설계한다
- 모바일에서 곡선/간격 리듬을 유지하되 장식 밀도를 줄인다

# Don't
- 네온/메탈릭/사이버 계열 톤을 혼합하지 않는다
- 강한 하드 섀도우나 각진 박스 스타일을 사용하지 않는다
- 패턴 대비를 높여 텍스트 가독성을 떨어뜨리지 않는다
- 모든 버튼을 동일 강도로 강조하지 않는다
- 우측 고정 대형 이미지로 정보 폭을 침범하지 않는다

# Notes
핵심은 “자연스러운 편안함 + 실사용성”의 균형이다. 감성은 배경과 재질에서, 명확함은 타이포/표/폼에서 만든다.
