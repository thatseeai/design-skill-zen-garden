# Intent
전통 한국 건축의 단청(丹靑)에서 영감을 받은 장식적이고 상징적인 인터페이스. 목재 위에 채색한 오방색의 대비, 반복되는 기하 문양, 균형 잡힌 대칭 구성을 현대 대시보드 UI로 재해석한다. 화면은 단청 처마 아래에 들어온 듯한 인상을 주되, 데이터 가독성과 조작성은 유지한다.

# Style Rules
- 오방색 기반 팔레트(청록, 적색, 황금, 백색, 먹색)를 사용하고 색 대비를 분명하게 유지한다
- 카드/섹션 경계는 단순한 라운드 대신 직선과 이중 보더로 처리해 목조 구조의 단단함을 표현한다
- 섹션 타이틀, 상단 바, 사이드바에 반복 문양(줄무늬, 원형, 연속 프레임)을 적용한다
- 배경은 평면 단색이 아니라 은은한 목재 톤과 문양 레이어를 중첩한다
- 버튼과 배지는 장식적이되 과도한 입체감은 피하고, 선명한 색 블록과 보더로 상태를 구분한다
- 본문 텍스트는 읽기 좋은 한글 서체를 우선하고, 제목은 전통적 분위기의 세리프 계열을 사용한다
- 여백은 충분히 확보해 장식 요소와 콘텐츠가 충돌하지 않도록 한다
- 호버/활성 상태는 색상 반전 또는 보더 강조 중심으로 일관되게 표현한다

# Tokens
- color.primary: #0f7a72 (청록)
- color.secondary: #b4382f (단청 적색)
- color.accent: #d39b2a (황금)
- color.ink: #1f2430 (먹색)
- color.paper: #f7f2e8 (한지 톤)
- color.wood: #8a5a3b (목재 톤)
- radius.sm: 2px
- radius.md: 4px
- radius.lg: 6px
- border.frame: 2px solid #d39b2a
- shadow.soft: 0 6px 18px rgba(31, 36, 48, 0.18)

# Do
- 반복 문양은 타이틀/프레임/구분선처럼 구조를 강조하는 위치에만 사용한다
- 주요 상호작용 요소는 청록/적색/황금의 조합으로 우선순위를 명확히 한다
- 섹션 제목에 장식 라인과 대비색 포인트를 넣어 페이지 리듬을 만든다
- 라이트/다크 모두 단청 분위기가 유지되도록 토큰 대비를 관리한다
- 모바일에서도 장식 요소가 콘텐츠를 가리지 않게 크기와 투명도를 조정한다

# Don't
- 네온톤, 파스텔톤, 그레이 단색 위주의 현대 미니멀 팔레트 사용 금지
- 큰 블러, 과도한 3D 그림자, 유리질(Glassmorphism) 효과 남용 금지
- 원형 라운드 UI를 광범위하게 적용해 전통 구조감을 약화시키지 않는다
- 장식 문양을 텍스트 가독성 위에 겹치지 않는다
- 서로 다른 시각 언어(사이버펑크, 스큐어모피즘 등)를 혼합하지 않는다

# Notes
단청 스타일의 핵심은 "색의 상징성 + 반복 질서 + 구조적 장식"이다. 장식은 배경이 아니라 정보 구조를 강조하는 장치로 사용하며, 모든 컴포넌트는 동일한 프레임 언어를 공유해야 한다.
