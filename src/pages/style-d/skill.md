# Intent
Bold, vibrant color-block design inspired by CSS Zen Garden's classic aesthetic. Emphasizes strong typographic hierarchy, generous whitespace, and high-contrast color combinations. Creates a cheerful, confident interface that feels both retro and modern.

# Style Rules
- Use bright teal/cyan (#75d1cd) as primary background color
- Apply bold magenta/pink (#cf3874) as primary accent for buttons and highlights
- Implement large, bold sans-serif typography with strong weight contrast
- Create distinct color blocks for different sections (teal, white, yellow, magenta)
- Use uppercase labels and headers for visual impact
- Maintain generous padding and spacing between elements
- Apply sharp corners (no border radius) for geometric clarity
- Use minimal shadows; rely on color contrast for depth
- Implement high contrast text (white on dark, dark on light)
- Table should have clean horizontal rules with alternating subtle backgrounds
- Sidebar should be dark navy/charcoal (#2b303b) for strong contrast
- Use yellow (#f8f4d3) blocks for secondary information panels

# Tokens
- color.primary: #75d1cd (teal)
- color.accent: #cf3874 (magenta)
- color.background: #ffffff (white)
- color.sidebar: #2b303b (dark navy)
- color.secondary: #f8f4d3 (light yellow)
- color.text.primary: #2b303b (dark)
- color.text.inverse: #ffffff (white)
- radius.none: 0px (sharp corners)
- space.section: 48px (generous)
- space.card: 32px
- typography.heading: 700+ weight
- typography.scale: 1.5-2.5x larger headings

# Do
- Use bold, confident typography with strong hierarchy
- Create distinct color zones for different content areas
- Maintain high contrast ratios for readability
- Use uppercase sparingly for labels and section headers
- Apply generous whitespace to let content breathe
- Keep geometric shapes sharp and clean
- Use color blocking to organize information
- Make interactive elements (buttons, links) visually prominent with color

# Don't
- Avoid rounded corners or soft edges
- Don't use gradients or subtle color transitions
- Avoid drop shadows or heavy effects
- Don't use muted or pastel colors
- Avoid tight spacing or cramped layouts
- Don't mix too many colors beyond the core palette
- Avoid small or thin typography for important elements
- Don't let background colors reduce text legibility
