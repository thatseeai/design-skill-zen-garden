# Intent
하이틴 만화 컷과 네온 스티커가 겹쳐진 Y2K 카와이 비주얼을 대시보드 UI로 번역한다. 전체 인상은 "달콤하지만 강한 전기감"이다. 핑크/시안/보라의 고채도 대비, 굵은 라인, 하프톤 패턴, 스파클 장식을 사용하되 정보 구조는 선명하게 유지해 장식성과 사용성을 동시에 확보한다.

# Style Rules
- 배경은 버블검 핑크에서 시안으로 흐르는 그라데이션을 기본으로 하고 하프톤 패턴을 얇게 겹친다
- 카드/탭/입력창은 이중 아웃라인과 소프트 글로우를 적용해 스티커 컷아웃 느낌을 만든다
- 라운드는 크게 사용하되 모든 컴포넌트의 라운드 체계를 통일한다
- 헤드라인은 개성 있는 디스플레이 폰트, 본문은 둥근 산세리프를 사용한다
- 포인트 컬러는 네온 핑크와 네온 시안을 중심으로 제한하고 보조색으로 보라를 사용한다
- 상태 배지는 텍스트 대비를 높게 유지하고 얇은 회색 배지는 피한다
- 장식 오브젝트(하트, 스파클)는 콘텐츠 레이어 뒤 또는 외곽에만 배치한다
- hover/focus 상태는 밝기 증가와 얕은 이동 효과로 통일해 반응성을 만든다
- 다크 모드에서는 동일한 색 관계를 유지하고 명도만 낮춰 형광 포인트를 살린다

# Do
- 숫자/제목/보조 텍스트의 계층을 확실히 나눠 정보 스캔 속도를 유지한다
- 액션 버튼은 선명한 대비와 충분한 크기로 시각적 우선순위를 명확히 둔다
- 장식 레이어는 `pointer-events-none`과 낮은 불투명도로 처리해 인터랙션을 방해하지 않게 한다
- 테이블과 폼의 경계선은 두께를 통일해 전체 시스템 일관성을 유지한다
- 모바일에서는 장식 요소를 줄여 콘텐츠 폭과 가독성을 우선한다

# Don't
- 네온 컬러를 무분별하게 늘려 화면 전체를 동일한 시각적 강도로 만들지 않는다
- 장식 패턴이 본문 텍스트 아래에 직접 깔리게 두지 않는다
- 컴포넌트마다 다른 보더 굵기/라운드를 사용해 시스템을 분산시키지 않는다
- 긴 본문 영역에서 저대비 보라 텍스트를 사용하지 않는다
- 글로우와 그림자를 과도하게 키워 블러 처리된 화면처럼 보이게 하지 않는다

# Notes
핵심은 "만화풍 감정 표현"과 "대시보드 실사용성"의 균형이다. 시선 유도는 강하게, 정보 해석은 빠르게 만드는 것이 목표다.
