# Intent
Y2K 글로시 UI를 현대 대시보드에 재해석한다. 크롬 같은 반사광, 젤리처럼 볼륨감 있는 버튼, 메탈릭 하이라이트를 중심으로 “디지털 오브제” 같은 시각 경험을 만든다. 화려하지만 정보 구조는 명확하게 유지한다.

# Style Rules
- 배경은 단색 금지: 그라데이션 + 반사광 레이어 + 미세 패턴을 함께 사용한다
- 카드/테이블/모달은 동일한 글로시 언어(하이라이트 라인, 내부 반사, 소프트 글로우)를 공유한다
- 버튼은 볼륨감 있게: 상단 하이라이트 + 하단 그림자 + 명확한 눌림 상태를 갖는다
- 상태 배지는 색상 의미가 즉시 드러나도록 채도/명도 대비를 충분히 확보한다
- 타이포는 미래지향 산세리프 기반으로 선명한 계층을 유지한다
- 코너는 중간 라운드(12~18px)로 디지털 오브제 느낌을 준다
- 주요 CTA만 강하게 강조하고 나머지는 보조 톤으로 낮춘다
- 다크/라이트 모두 동일한 재질감 인상을 유지한다

# Do
- 컴포넌트 전반에서 동일한 하이라이트/보더 두께 체계를 유지한다
- 콘텐츠 가독성을 위해 본문 텍스트 대비를 충분히 확보한다
- 인터랙션 상태(hover/active/focus)를 명확한 광택 변화로 표현한다
- 모바일에서 장식 레이어 강도를 낮춰 정보 영역을 우선한다
- CSS 중심으로 구현해 성능 부담을 최소화한다

# Don't
- 장식 이미지 한 장에 의존해 분위기를 만들지 않는다
- 과도한 블러/필터 남용으로 텍스트를 흐리게 만들지 않는다
- 모든 버튼을 1차 CTA처럼 강조하지 않는다
- 상태 색을 비슷한 톤으로 묶어 의미 구분을 약화시키지 않는다
- 배경 레이어를 강하게 적용해 표/폼 가독성을 해치지 않는다

# Notes
핵심은 “유광 재질감의 일관성”이다. 개별 요소보다 시스템 전반의 동일한 글로시 규칙이 중요하다.
