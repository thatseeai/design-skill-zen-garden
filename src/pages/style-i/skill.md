# Intent
프리미엄 고객지원 랜딩 페이지에서 보이는 미니멀 에디토리얼 레이아웃을 대시보드 UI에 적용한다. 넓은 여백, 큰 라운드 카드, 절제된 회색 톤, 선명한 블루 링크 포인트를 중심으로 신뢰감 있고 정돈된 경험을 만든다.

# Style Rules
- 배경은 매우 옅은 쿨그레이를 사용하고, 주요 콘텐츠는 큰 반경의 밝은 카드로 분리한다
- 대비는 강한 색 다변화 대신 명도 차이로 만든다 (배경/카드/텍스트 3단계)
- CTA 성격의 인터랙션은 단일 블루 포인트 컬러로 통일한다
- 카드 내부 타이포는 큰 헤드라인 + 편안한 본문 계층을 유지한다
- 보더는 옅고 얇게, 그림자는 넓고 약하게 사용해 부유감을 준다
- 복잡한 장식 문양 대신 단순한 그라데이션 패널과 소프트 블러를 사용한다
- 모서리는 전체적으로 부드럽고 크게 라운딩한다
- 데이터 영역(테이블/폼)은 과도한 색 대신 중성 톤에서 상태 색만 제한적으로 사용한다

# Tokens
- color.background: #f5f5f7
- color.surface: #ffffff
- color.surface-muted: #f2f2f4
- color.text.primary: #1d1d1f
- color.text.secondary: #424245
- color.link: #06c
- color.border: #d2d2d7
- color.accent.blue: #4fa3ff
- radius.card: 28px
- radius.control: 14px
- shadow.soft: 0 12px 30px rgba(17, 17, 26, 0.08)

# Do
- 섹션을 넓고 큰 카드 단위로 그룹핑한다
- 헤드라인은 명확하게 크게, 설명은 충분한 줄간격으로 배치한다
- 블루 포인트는 링크/주요 액션에만 제한적으로 사용한다
- 스크롤 구간마다 시선이 쉬는 여백을 확보한다
- 라이트 모드의 정갈함을 유지하되 다크 모드에서도 대비를 유지한다

# Don't
- 강한 채도의 다색 조합 사용 금지
- 두꺼운 보더/강한 입체 그림자 남용 금지
- 불필요한 아이콘/패턴으로 시각적 복잡도 증가 금지
- 모든 버튼에 동일한 시각적 무게를 부여하지 않는다
- 타이포 크기를 고르게 맞춰 계층을 흐리지 않는다

# Notes
목표는 "제품 소개형 카드 레이아웃 + 서비스 안내형 타이포"의 균형이다. 콘텐츠를 카드처럼 큼직하게 묶고, 블루 링크를 리듬 포인트로 사용해 안내 중심 UI를 구현한다.
