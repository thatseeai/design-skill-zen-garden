# Intent
B2B SaaS 대시보드를 위한 미니멀하고 전문적인 디자인. 정보 밀도가 높고 효율적인 작업 환경을 제공하며, 불필요한 장식을 배제하고 기능에 집중한다. 차분하고 신뢰감 있는 톤으로 비즈니스 사용자에게 적합한 인터페이스를 구현한다.

# Style Rules
- 그림자는 최소화하고, 보더(1px solid)로 요소를 구분한다
- border-radius는 작게 유지한다 (2-4px)
- 색상은 저채도 팔레트를 사용한다 (그레이 베이스)
- 테이블은 고밀도로 구성하여 한 화면에 더 많은 정보를 표시한다
- 타이포그래피 계층을 명확히 하여 스캔이 용이하게 한다
- 여백은 타이트하게 유지하여 정보 밀도를 높인다
- 액션 버튼은 명확하되 절제된 스타일로 표현한다
- 배지는 단색 배경보다 아웃라인 스타일을 선호한다
- 포커스 링은 명확하게 보이도록 한다
- 호버 상태는 미묘한 배경색 변화로 표현한다

# Tokens
- color.primary: slate-900
- color.accent: blue-600
- color.success: green-600
- color.warning: amber-600
- color.error: red-600
- radius.sm: 2px
- radius.md: 4px
- space.section: 16px
- space.element: 8px

# Do
- 정보 계층을 명확히 구분한다
- 테이블과 리스트에서 밀도를 높게 유지한다
- 일관된 보더 스타일을 사용한다
- 액션 요소는 명확하게 구분한다
- 포커스 상태를 항상 표시한다

# Don't
- 화려한 그라디언트나 그림자 사용 금지
- 과도한 색상 사용 금지
- 라운드 코너를 크게 하지 않는다
- 장식적 요소 추가 금지
- 애니메이션을 과도하게 사용하지 않는다

# Dark Mode
- 다크 모드에서도 동일한 미니멀 원칙을 유지한다
- 저채도 다크 팔레트를 사용한다 (진한 회색/슬레이트 톤)
- 보더와 구분선의 대비를 적절히 유지한다
- 배지의 아웃라인 색상을 다크 모드에 맞게 밝게 조정한다
- 호버 상태의 배경색 변화를 미묘하게 유지한다
- 그림자는 다크 모드에서도 최소화한다

# Notes
전문적인 비즈니스 환경에서 사용하기 적합한 디자인. 정보 중심의 레이아웃으로 데이터 분석과 관리 작업에 최적화되어 있다. 차분한 색상과 명확한 구조로 장시간 사용해도 피로감이 적다.
