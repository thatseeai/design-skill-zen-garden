# Intent
겨울 럭셔리 아우터 편집숍의 화보 분위기를 대시보드 UI로 번역한다. 핵심은 "털 옷의 촉감"을 시각적으로 전달하는 것이다. 부드러운 크림/카멜/브라운 스펙트럼, 깊은 명암, 긴 곡선 라운드, 벨벳 같은 대비를 통해 포근하지만 고급스러운 인상을 만든다.

# Style Rules
- 배경은 밝은 크림 톤, 주요 섹션은 모피 질감 이미지가 섞인 카드로 구성한다
- 헤드라인은 클래식한 세리프, 본문은 읽기 쉬운 산세리프로 조합한다
- 카드/버튼/입력 요소는 중간 이상의 라운드를 사용해 부드러운 감각을 강화한다
- 포인트 색은 카멜 골드 계열을 중심으로 제한해 과도한 색 분산을 피한다
- 표면은 평면 단색보다 미세한 그라데이션과 텍스처를 사용한다
- 상태 배지는 톤온톤으로 표현하고 채도 높은 네온 계열은 배제한다
- 섀도우는 부드럽고 넓게, 테두리는 얇고 따뜻한 중립색으로 처리한다
- 다크 모드에서도 텍스트 대비를 우선해 가독성을 최상위로 유지한다

# Do
- 실제 퍼 텍스처 느낌의 이미지를 장식 레이어로 활용한다
- 카드 내부 계층(제목-수치-설명)을 명확히 유지한다
- 인터랙션 상태는 밝기 변화/미세한 그림자로 일관되게 처리한다
- 모바일에서도 텍스처 이미지가 콘텐츠를 가리지 않게 투명도를 조절한다
- 버튼/탭의 대비를 충분히 높여 조작 가능성을 명확히 보여준다

# Don't
- 차가운 블루/네온 계열을 주요 포인트로 사용하지 않는다
- 과도한 글래스모피즘이나 고채도 그라데이션을 사용하지 않는다
- 카드 배경과 텍스트 명도를 비슷하게 두지 않는다
- 텍스처 이미지를 전체 화면에 강하게 깔아 가독성을 해치지 않는다
- 모든 요소를 동일한 시각적 무게로 만들어 계층을 흐리지 않는다

# Notes
이 스타일의 핵심은 "부드러운 질감 + 고급스러운 명도 대비"다. 이미지 텍스처는 장식이 아니라 분위기와 맥락을 만드는 보조 레이어로만 사용한다.
