# Intent
레트로 문구 스티커가 잔뜩 붙은 파스텔 패키지 감성을 대시보드 UI에 적용한다. 비닐 포장지 위에 둥근 스티커 오브젝트가 레이어링된 분위기를 통해 유쾌하고 친근한 감정 경험을 만든다. 정보는 충분히 읽히되, 전체 인상은 '귀엽고 장난기 있는 POP 디자인'이 되도록 한다.

# Style Rules
- 전체 배경은 라이트 핑크/라일락 기반으로 유지하고 카드는 스티커처럼 도드라지게 배치한다
- 카드/버튼/탭은 두꺼운 외곽선 또는 이중 아웃라인으로 스티커 컷팅 느낌을 준다
- 모서리는 크게 라운딩해 유연하고 장난스러운 형태를 유지한다
- 타이포는 둥근 산세리프 중심으로, 헤드라인은 굵고 큼직하게 구성한다
- 장식은 별, 구름, 꽃, 말풍선 같은 단순 모티프를 반복해 리듬을 만든다
- 보조 정보의 대비를 충분히 확보해 '귀엽지만 읽기 어려운 UI'를 피한다
- 상태 색상은 파스텔 그린/옐로/코랄 계열로 통일한다
- 다크 모드에서는 동일 구조를 유지하되 채도와 명도 대비를 올려 가독성을 보장한다

# Do
- 스티커 스타일 요소(두꺼운 테두리, 밝은 배경, 그림자)를 일관되게 적용한다
- 제목/숫자/설명의 시각적 계층을 명확하게 나눈다
- 장식 이미지는 콘텐츠 뒤에 두고 불투명도를 낮춰 가독성을 보호한다
- 인터랙션 시 미세한 밝기 변화와 이동(translateY)을 사용한다
- 모바일에서 장식 요소를 줄여 주요 정보 폭을 확보한다

# Don't
- 네온 고채도 컬러를 과도하게 사용하지 않는다
- 얇은 회색 텍스트를 파스텔 배경 위에 그대로 두지 않는다
- 장식 스티커가 버튼/폼/테이블 텍스트를 가리지 않게 한다
- 각 컴포넌트의 보더 두께와 라운드 값을 제각각 쓰지 않는다
- 그림자를 너무 진하게 넣어 무거운 인상을 만들지 않는다

# Notes
목표는 "패키지 디자인 감성 + 실제 대시보드 사용성"의 균형이다. 예쁜 장식보다 정보 명확성이 항상 우선이다.
