# Intent
미래형 브루탈리즘(Futuristic Brutalism)으로 화면 전체를 재구성한다. 매트한 딥 블랙 바탕 위에 네온 컬러를 날카롭게 배치하고, 큰 타이포와 비대칭 블록 구조로 즉시 강한 시각적 인상을 만든다. 데이터는 기계적이고 선명하게 읽히되, 인터페이스는 공격적이고 실험적인 아트 방향을 유지한다.

# Style Rules
- 배경은 단색 금지: 그리드, 노이즈, 네온 글로우 레이어를 반드시 중첩한다
- 카드/테이블/폼/탭/버튼은 직각 중심(거의 0 radius)으로 통일한다
- 주요 요소는 두꺼운 보더와 강한 대비로 정보 우선순위를 드러낸다
- 타이포는 굵고 큰 헤드라인을 사용하고 본문은 압축된 리듬으로 구성한다
- 네온 포인트 컬러는 2~3개로 제한해 시스템 일관성을 유지한다
- 비대칭 그리드 감각을 위해 여백/정렬 축을 일부 의도적으로 깨뜨린다
- 상태 색상은 의미가 즉시 전달되도록 대비를 극대화한다
- 다크 모드에서도 텍스트 대비를 최우선으로 유지한다

# Do
- 헤드라인, KPI 숫자, CTA에서 네온 포인트를 전략적으로 반복한다
- 상호작용 요소는 hover/focus에서 명확한 색 변화와 외곽선으로 피드백 준다
- 모바일에서는 비대칭 느낌을 유지하되 터치 영역과 가독성을 우선한다
- 테이블/폼 텍스트 대비를 높여 실사용성을 확보한다
- 섹션마다 블록 구조가 분명하게 보이도록 패널 분리를 유지한다

# Don't
- 파스텔/저대비 컬러를 중심 팔레트로 사용하지 않는다
- 둥근 카드/소프트 UI 스타일로 브루탈리즘 톤을 약화시키지 않는다
- 장식 이미지 한 장을 특정 위치에 크게 고정하지 않는다
- 과도한 애니메이션으로 정보 흐름을 방해하지 않는다
- 배경 장식이 텍스트 가독성을 침범하지 않게 한다

# Notes
핵심은 “기계적 질서 + 의도적 불균형”이다. 강한 대비와 블록 단위 구성으로 즉시 새로운 스타일로 인지되어야 한다.
