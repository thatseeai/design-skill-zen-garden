# Intent
레트로 스티커 팝 스타일을 현대 대시보드에 적용한다. 90년대 문구/패키지의 활기 있는 컬러, 두꺼운 외곽선, 하프톤 프린트 질감을 결합해 즉각적으로 밝고 재미있는 인상을 만든다. 감성은 과감하지만 정보 구조는 선명하게 유지한다.

# Style Rules
- 파스텔 고채도 조합(레드/시안/옐로/민트)을 핵심 팔레트로 사용한다
- 카드/탭/버튼/배지/모달은 두꺼운 아웃라인과 큰 라운드로 스티커 컷팅 느낌을 준다
- 배경은 단색 금지: 하프톤 도트, 스트라이프, 소프트 그라데이션을 레이어링한다
- 제목은 크고 굵은 라운드 산세리프, 본문은 대비 높은 중간 굵기로 유지한다
- 섹션 사이 여백을 넉넉히 두고 장식 요소는 주변부에만 배치한다
- 상태 색상은 파스텔톤 내에서 구분 가능하도록 명도/채도 차이를 명확히 둔다
- 다크 모드에서도 카드/본문 대비를 유지해 읽기 어려워지지 않게 한다
- 숫자(KPI)는 tabular-nums를 적용해 정렬 안정성을 확보한다

# Do
- 컴포넌트 전반에서 보더 두께/라운드/그림자 규칙을 통일한다
- 배경 패턴은 저강도로 유지해 콘텐츠 가독성을 우선한다
- 주요 액션 버튼과 활성 탭은 강한 대비로 즉시 구분 가능하게 한다
- 모바일에서 장식 요소를 축소/숨김 처리해 폭을 확보한다
- 테이블/폼 텍스트는 장식보다 정보 전달이 우선되게 유지한다

# Don't
- 카드 내부에 과도한 장식 그래픽을 넣어 텍스트를 방해하지 않는다
- 저대비 파스텔 텍스트를 본문에 사용하지 않는다
- 우측 고정 대형 이미지로 정보 영역 폭을 줄이지 않는다
- 컴포넌트마다 다른 시각 언어를 혼합하지 않는다
- 그라데이션/패턴을 진하게 넣어 눈 피로를 유발하지 않는다

# Notes
핵심은 "장난스러운 스타일 + 실사용 가독성"의 균형이다. 시각적 즐거움은 배경과 외곽 스타일에서 만들고, 데이터 영역은 항상 또렷하게 보이게 설계한다.
