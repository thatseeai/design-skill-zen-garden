# Intent

한국 설날(Seollal)의 격조 있는 명절 분위기를 현대 UI에 녹인 스타일.
한지(韓紙)의 따뜻한 크림 질감, 단청 오방색에서 추출한 절제된 컬러 팔레트,
세배·복주머니·매화·전통 문양이 연상되는 장식 요소를 전문적 타이포그래피와
여백 활용으로 정리한다. "할머니 댁의 설날 아침"이 아닌,
"디자인 서적에 실릴 법한 정제된 설날"을 목표로 한다.

# Style Rules
- 팔레트는 한지 크림(#F5EFE0)을 바탕으로, 단홍(#C83E3E), 남청(#2B4C7E), 금박(#C5A55A), 소나무 녹(#3B6B4F)을 포인트로 사용
- 카드·패널 배경은 한지 질감(미세 섬유 노이즈)을 은은하게 적용
- 보더는 단정하고 얇되, 꼭짓점·헤더에 전통 문양(수(壽)·만(卍) 연속 패턴)을 간결히 암시
- radius는 중간(10–14px) — 너무 각지지도, 너무 둥글지도 않게
- 테이블·폼은 높은 가독성 유지: 밀도보다 정갈한 여백
- 버튼 1차 CTA는 단홍 배경+크림 텍스트, 보조 버튼은 남청 보더
- 상태 배지는 오방색 응용(활성=녹, 비활성=회, 대기=금)
- 그림자는 따뜻한 톤(#5C3A1A 계열) 저강도로 사용
- heading 폰트는 Noto Serif KR 또는 유사 세리프, body는 Pretendard/Inter
- 다크 모드: 짙은 먹색(#1A1714) 배경에 한지 톤 밝기 유지, 금박·단홍 채도 살짝 올림

# Do
- 매화 가지, 복주머니, 전통 격자 문양을 **데코레이션 요소로만** 사용(구조 침범 금지)
- 금박색은 제목 장식·배지·아이콘에 포인트로 활용
- heading과 body 서체 대비로 격식 있는 계층 구성
- 충분한 여백과 정렬로 "정갈함" 연출
- 다크 모드에서 금박·단홍이 더 또렷하게 빛나도록

# Don't
- 과도한 전통 일러스트·한복 이미지 삽입 금지 (CSS/SVG 패턴으로 절제)
- 원색 레드·블루를 그대로 쓰지 말 것 — 반드시 한국 전통색 톤으로 조율
- 네온·글로시·메탈릭 효과 금지
- 전통 문양이 텍스트 가독성을 방해하지 않도록
- "새해 복 많이 받으세요" 같은 문구를 콘텐츠에 삽입하지 말 것 (스타일만 변경)

# Notes
핵심은 "한지 위의 정갈함 + 전통색의 절제된 화려함".
설날의 따뜻한 기억과 격식이 공존하는 인터페이스.
